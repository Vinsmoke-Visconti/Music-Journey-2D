# 🎵 Music Journey 2D

**Music Journey 2D** is a sophisticated web-based music visualizer that blends high-performance 2D graphics with real-time audio analysis. Users experience a dynamic journey where the environment and vehicle physics react instinctively to the music's rhythm.

## 🚀 Key Features

- **Real-time Audio Analysis**: Utilizes Web Audio API and FFT (Fast Fourier Transform) to sync visual elements with music frequencies (Bass, Mid, Treble).
- **Advanced Vehicle Physics**: Features a custom-built 2D physics engine with suspension simulation and ground detection, allowing vehicles to follow road curvatures realistically.
- **Dynamic Environments**: Multiple themes (Beach, Desert, Snow) with parallax scrolling and procedural road generation.
- **Time-Sync Effects**: Sky gradients and lighting conditions interpolate based on song progress.
- **Data-Driven Architecture**: Fully configurable vehicle stats and environment parameters using TypeScript.

## 🛠 Tech Stack

- **Core Engine**: [PixiJS](https://pixijs.com/) (WebGL-based high-performance 2D rendering).
- **Frontend**: Vite, TypeScript, HTML5/CSS3.
- **Audio Processing**: Web Audio API.
- **Animation**: GSAP (GreenSock Animation Platform) for smooth transitions.
- **Backend (Phase 4)**: Supabase/Keycloak for authentication and user data storage.
